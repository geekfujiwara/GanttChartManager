# アプリケーションロゴ・アイコンガイド

## ロゴバリエーション

### 1. メインロゴ (`logo.svg`)
- **用途**: メインアプリケーションロゴ、ヘッダー表示
- **サイズ**: 64x64px
- **デザイン**: Power Appsブルー背景、ガントチャート要素、ライトニングシンボル
- **特徴**: シンプルでスケーラブル、コードブラケット `< >` 付き

### 2. ファビコン (`favicon.svg`)
- **用途**: ブラウザタブアイコン
- **サイズ**: 32x32px
- **デザイン**: 簡略化されたガントチャート要素
- **特徴**: 小さなサイズでも視認性を保つ

### 3. 高解像度ロゴ (`logo-256.svg`)
- **用途**: アプリストア、大きな表示領域
- **サイズ**: 256x256px
- **デザイン**: グラデーション、詳細なガントチャート
- **特徴**: 高品質、アイコン要素豊富

### 4. ガントチャート専用ロゴ (`logo-gantt.svg`)
- **用途**: ガントチャート画面、専門的な表示
- **サイズ**: 128x128px
- **デザイン**: 詳細なプロジェクト進捗表現
- **特徴**: 依存関係の矢印、進捗インジケーター付き

## デザイン要素

### カラーパレット
- **Primary Blue**: `#0078D4` (Power Apps標準色)
- **Secondary Blue**: `#005A9E` (境界線)
- **Accent Gold**: `#FFD700` (ライトニング、マーカー)
- **Status Colors**:
  - 完了: `#22C55E` (緑)
  - 進行中: `#F59E0B` (オレンジ)
  - 遅延: `#EF4444` (赤)
  - 未開始: `#8B5CF6` (紫)

### シンボル要素
- **ガントチャートバー**: プロジェクトタスクを表現
- **ライトニング**: Power Appsのシンボル
- **コードブラケット**: Code Appsのアイデンティティ
- **進捗インジケーター**: タスクの完了状況
- **タイムライン**: プロジェクトスケジュール

## 使用方法

### HTML内での参照
```html
<!-- ファビコン -->
<link rel="icon" type="image/svg+xml" href="/assets/favicon.svg" />

<!-- アプリアイコン（Apple Touch Icon） -->
<link rel="apple-touch-icon" sizes="192x192" href="/assets/logo-256.svg" />
```

### React コンポーネント内での使用
```tsx
<img 
  src="/assets/logo.svg" 
  alt="Gantt Chart App Logo" 
  className="h-8 w-8"
  onError={(e) => {
    // フォールバック処理
    console.warn('Logo SVG not found');
    (e.target as HTMLImageElement).style.display = 'none';
  }}
/>
```

### Power Apps設定
```json
{
  "logoPath": "./public/assets/logo.svg"
}
```

## アクセシビリティ考慮

1. **Alt テキスト**: 適切な代替テキストを設定
2. **コントラスト**: WCAG 2.1 AA準拠の色彩設計
3. **スケーラビリティ**: SVGベースでリサイズ対応
4. **フォールバック**: ロード失敗時の適切なハンドリング

## ファイル構造

```
public/assets/
├── logo.svg           # メインロゴ (64x64)
├── favicon.svg        # ファビコン (32x32)
├── logo-256.svg       # 高解像度版 (256x256)
└── logo-gantt.svg     # ガントチャート専用 (128x128)
```

## カスタマイズガイド

### カラー変更
各SVGファイル内の`fill`属性と`stroke`属性を変更することで、企業ブランドに合わせたカスタマイズが可能です。

### サイズ調整
SVGファイルの`viewBox`属性を調整することで、アスペクト比を保ったまま表示サイズを変更できます。

### 要素の追加・削除
企業ロゴやプロジェクト固有のシンボルを追加する場合は、既存の要素を参考に新しい`<g>`要素として追加してください。
